# 🎨 圖像生成應用設計：旅遊規劃教學四步驟插畫

本設計案例以「旅遊規劃教學」為主題，利用 Fooocus 產生三張具代表性的插畫圖像，模擬應用於旅遊文章、社群貼文與簡報封面等情境。

使用「」模型，每一張圖像修改 **Prompt**\，以呈現相同風格下的多樣表現  
圖像皆屬於日系插畫風格，搭配溫和配色與教學導向的情境設計
